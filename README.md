# 📚 TS Academy Library API

A production-style RESTful API for managing a school library system built with Node.js, Express, and MongoDB.

This project demonstrates backend engineering skills including MVC architecture, MongoDB relationships, pagination, search, and a full book borrowing/return system.

---

## 🚀 Features

### 📖 Core Modules
- Authors management (CRUD)
- Books management (CRUD)
- Students management
- Library attendants management

### ⚙️ Advanced Features
- Book borrowing & return system
- Overdue detection (return date tracking)
- Pagination for books listing
- Search books by title or author
- Duplicate ISBN prevention
- MongoDB population relationships
- RESTful API architecture
- Swagger API documentation

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- Swagger (OpenAPI 3.0)
- dotenv
- ES Modules (Modern JavaScript)

---

## 📁 Project Structure
/models → Database schemas
/controllers → Business logic
/routes → API endpoints
/middleware → Validation & helpers
/config → Database & Swagger config
server.js → App entry point
